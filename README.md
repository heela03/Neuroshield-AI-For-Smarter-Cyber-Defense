# 🛡️ NeuroShield IDS  
### AI-Powered Intrusion Detection with Explainable Intelligence

---

## 🚀 Overview

**NeuroShield IDS** is an advanced AI-based Intrusion Detection System that detects, analyzes, and explains cyber threats in network traffic. Unlike traditional IDS solutions, this system integrates **Explainable AI (XAI)** to make predictions transparent and actionable.

The system combines **deep learning, SHAP explainability, MITRE ATT&CK mapping, risk analysis, and automated reporting** to deliver a complete cybersecurity solution.

---

## 🎯 Features

### 🔍 Intelligent Threat Detection
- Binary classification (Normal vs Attack)
- Multiclass classification (Attack type detection)
- Deep learning–based model trained on UNSW-NB15 dataset

### 🧠 Explainable AI (XAI)
- SHAP-based feature importance analysis
- Global and local explanations
- Improves trust and interpretability

### 🗺️ MITRE ATT&CK Mapping
- Maps detected attacks to standardized tactics & techniques
- Helps understand attacker behavior

### 💡 Recommendation System
- Suggests mitigation strategies based on attack type
- Enables proactive defense

### ⚠️ Risk Analysis
- Classifies threats into:
  - Low
  - Medium
  - High
- Helps prioritize critical threats

### 📊 Interactive Dashboard
- Built using Streamlit
- Displays:
  - Detection results
  - SHAP explanations
  - Risk levels
  - Recommendations

### 📄 Automated Report Generation
- Generates structured reports
- Export formats: CSV / Excel
